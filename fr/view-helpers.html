View helpers

itemSetsTree

Le view helper itemSetsTree met à disposition les méthodes suivantes:

getRootItemSets

Retourne les collections mères (les collections qui n’ont pas de parent).

Synopsis

<?php $itemSets = $this->itemSetsTree()->getRootItemSets($options); ?>

Arguments

$options

A tableau associatif qui peut contenir les entrées suivantes:

sorting_method

Méthode de tri. Peut être 'none' (aucun tri) ou 'title' (tri par titre). Si non renseigné, la configuration globale est utilisée.

Retourne

Un tableau de Omeka\Api\Representation\ItemSetRepresentation

Exemples

<?php $itemSets = $this->itemSetsTree()->getRootItemSets(); ?>
<?php $itemSets = $this->itemSetsTree()->getRootItemSets(['sorting_method' => 'title']); ?>

getItemSetsTree

Retourne les collections dans une structure hiérarchique

Synopsis

<?php $itemSets = $this->itemSetsTree()->getItemSetsTree($maxDepth, $options); ?>

Arguments

$maxDepth

Profondeur maximale de l’arbre. Si non renseigné ou null, retourne l’arbre complet.

$options

A tableau associatif qui peut contenir les entrées suivantes:

site_id

Site identifier. If given, this site’s configuration will be used. Defaults to the current site.

sorting_method

Méthode de tri. Peut être 'none' (aucun tri) ou 'title' (tri par titre). Si non renseigné, la configuration globale est utilisée.

Retourne

Un tableau de tableaux associatifs. Chaque tableau associatif représente un nœud de l’arbre et contiendra les clés suivantes:

itemSet

L’objet Omeka\Api\Representation\ItemSetRepresentation correspondant au nœud de l’arbre.

children

Un tableau de tableaux associatifs représentant les enfants du nœud de l’arbre

Exemple:

<?php
[
    [
        'itemSet' => $itemSet1, // ItemSetRepresentation
        'children' => [],
    ],
    [
        'itemSet' => $itemSet2, // ItemSetRepresentation
        'children' => [
            [
                'itemSet' => $itemSet3, // ItemSetRepresentation
                'children' => [],
            ],
        ],
    ],
]
?>

Exemples

<?php
// returns the whole tree
$itemSets = $this->itemSetsTree()->getItemSetsTree();

// returns only the root item sets
$itemSets = $this->itemSetsTree()->getItemSetsTree(1);

// returns an item sets tree corresponding to the current site
$itemSets = $this->itemSetsTree()->getItemSetsTree(null, [
    'site_id' => $this->layout()->site->id(),
]);

// returns an item sets tree corresponding to the current site, limited to
// two levels, sorted by title
$itemSets = $this->itemSetsTree()->getItemSetsTree(2, [
    'site_id' => $this->layout()->site->id(),
    'sorting_method' => 'title',
]);
?>

getParent

Retourne le parent d’une collection.

Synopsis

<?php $parentItemSet = $this->itemSetsTree()->getParent($itemSet); ?>

Arguments

$itemSet

Un objet de type Omeka\Api\Representation\ItemSetRepresentation

Retourne

Un objet de type Omeka\Api\Representation\ItemSetRepresentation

getAncestors

Retourne les ancêtres d’une collection.

Synopsis

<?php $ancestors = $this->itemSetsTree()->getAncestors($itemSet); ?>

Arguments

$itemSet

Un objet de type Omeka\Api\Representation\ItemSetRepresentation

Retourne

Un tableau d’objets de type Omeka\Api\Representation\ItemSetRepresentation.

Le premier élément sera le parent, le deuxième élément sera le grand-parent, et ainsi de suite.

getChildren

Retourne les enfants d’une collection.

Synopsis

<?php $children = $this->itemSetsTree()->getChildren($itemSet, $options); ?>

Arguments

$itemSet

Un objet de type Omeka\Api\Representation\ItemSetRepresentation

$options

A tableau associatif qui peut contenir les entrées suivantes:

sorting_method

Méthode de tri. Peut être 'none' (aucun tri) ou 'title' (tri par titre). Si non renseigné, la configuration globale est utilisée.

Retourne

Un tableau d’objets de type Omeka\Api\Representation\ItemSetRepresentation.

getDescendants

Retourne les descendants d’une collection.

Synopsis

<?php $descendants = $this->itemSetsTree()->getDescendants($itemSet); ?>

Arguments

$itemSet

Un objet de type Omeka\Api\Representation\ItemSetRepresentation

Retourne

Un tableau d’objets de type Omeka\Api\Representation\ItemSetRepresentation.
