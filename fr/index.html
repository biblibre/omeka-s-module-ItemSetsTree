Item Sets Tree

Item Sets Tree est un module pour Omeka S. Il permet d’organiser les collections en arborescence.

Installation

Voir la documentation utilisateur générale pour installer un module

Comment ça marche

Une fois le module installé, vous pouvez définir une collection parent au moment de la création ou de la modification d’une collection.

Vous pouvez visualiser l’arborescence complète en cliquant sur l’entrée de menu dans la barre latérale gauche.

L’arborescence est aussi accessible dans les sites publics. Vous pouvez ajouter un lien de navigation vers l’arborescence dans Sites » Navigation.
